## A Bomberman-style game with multiplayer option.

Fork from https://github.com/DmytroVasin/bomber

### Installation
```bash
npm install
```

### Run
```bash
npm start
```

### Build tool for windows
In order to be able to edit front end files
```bash
pkg -t node18-win -o client/exposer.exe .
cd client
./exposer.exe
```
